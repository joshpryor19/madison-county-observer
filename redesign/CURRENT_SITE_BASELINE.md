# Current Live-Site Baseline

Observed July 17, 2026. The repository remains the source of truth; Claude Code must verify this inventory before editing.

## Current public pattern

- A global establishment/tagline line.
- Text masthead and primary navigation.
- `FREE — TAKE ONE (WE'RE WATCHING)` issue language.
- Atmospheric/weather ticker.
- Front page grouped by Town Hall Dispatches, Matters of Commerce, Society Pages, and Crime Blotter.
- Individual article routes use filenames such as `article8.html` and `article14.html`.
- Article pages currently contain breadcrumb, headline, deck, byline/category metadata, hero image, simple photograph credit, article body, developing-story line, Letters to the Editor prompt, and footer.
- About page contains the anonymous masthead and the strongest institutional lore.
- Contact currently relies on a direct-email/mail-client workflow.

## Highest-priority gaps to address

1. Direct article visitors need related-story paths and a reason to explore.
2. Hero images need accurate evidence classifications and a shared file-strip system.
3. The homepage needs a lead/secondary hierarchy rather than equal visual weight.
4. Newsletter and field-report integration points should be designed but not falsely activated.
5. Repeated header/footer/template code should be centralized only when the existing stack safely supports it.
6. The unowned eye should be removed from long-term brand use and replaced later with an original asset.

## Preserve

- Current article copy.
- Current public URLs.
- Existing section names and strongest brand language.
- Google Analytics configuration.
- Visible satire disclosure.
